### Структура решения
Заданий много, времени мало, поэтому необходимо понимать, как нужно решать каждое из этих заданий.

> Следующее разбиение задач использует автор и не обязывает вас его использовать

Давайте разобьем задачи на 3 группы:
1. Задачи на код (1, 2, 5, 6, 8, 9, 13, 14, 15, 16, 17, 19-21, 23, 24, 25, 26, 27)
2. Задачи на Excel (3, 18, 22, 27)
3. Задачи о медитировании над листочком (4, 7, 11, 12)

Задание 10 лучше выполнять в Word, но при большом желании эту задачу можно сделать с помощью Python.

Задание 27 можно решать 3 способами: Python, Excel, смешанный (Python + Excel)

### Код
Для ЕГЭ по информатике рекомендуется использовать язык программирования Python, так как при его использовании не нужно задумываться о переполнении целочисленных типов данных и о скорости работы. Также в Python есть множество встроенных библиотек, которые можно использовать на ЕГЭ.

Большую часть задач будем решать кодом, поэтому необходимо владеть ЯП на определённом уровне.
